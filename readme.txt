Easily embed Social Media content from Stackla into your Wordpress websites. 

== Description ==
Stackla is a social content marketing platform used by hundreds of global brands, agencies, media companies, and nonprofits to discover, curate and publish the best content from the social web. 

Stackla's (Simple Embed) Wordpress plugin allows you to display social media content and display via a Stackla widget directly on a Post or a Page.

<strong>Shortcode</strong>

>[stackla widget_id="WIDGET_ID"]
>[stackla widget_id="WIDGET_ID" filter_id="FILTER_ID"]
>[stackla widget_id="WIDGET_ID" tile_id="TILE_ID"]

== Installation ==
Download the plugin zip file, unzip it, and upload it to your Wordpress site under the /wp-includes/plugins directory.

Once installed, simply create a Widget and then insert the above Shortcode into any post or page. 


== Frequently Asked Questions ==

= Do I need a Stackla account? =
Yes. A Stackla account is required to display social content using this plugin. More information is available at [Stackla.com](https://stackla.com "Stackla Website")

= Which social networks are supported? =
Stackla supports 24 aggregation from networks out of the box.  Any content successfully aggregated into the Platform can be rendered through this Plugin.
